== Installation ==
Ensure you have actual version of WooCommerce plugin installed. Plugin supports and tested on WP 6.7.2 and PHP 7.4.33, WooCoomerce versions from 3.0.1 to 9.8.2
Upload the plugin files to the /wp-content/plugins/(or upload the plugin zip file using wp-admin.)
Activate the plugin through the 'Plugins' menu.
== Configuration ==
Go to Woocommerce -> Settings -> Payment.
Go to tab "hutko". If it doesn't exist - activate the plugin.
Allow this payment method (Enable). Name it "Online Payments".
Fill out the fields Merchant ID and Secret Key. You can get them on the portal.hutko.org
Customize other plugin settings if it`s necessary.
Save settings.
